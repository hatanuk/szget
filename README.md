# szget
everything you ever wanted from wget but less  
made after a strike of divine inspiration from reading [Beej's Guide to Network Programming](https://beej.us/guide/bgnet/pdf/bgnet_usl_c_1.pdf)

### Usage
szget url [-o filename]

### Installation
1. clone and cd into repo  
2. run `make` in terminal  
3. `szget` should be compiled. run it via `./szget`

### Notes
only supports HTTP or HTTPS schemas
